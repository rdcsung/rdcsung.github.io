# Install Ubuntu
1. BIOS configuration

On DELL, push F2 during start up of the computuer.

# Install gnome-shell, terminator
1. gnome-shell
> sudo apt-get install ubuntu-gnome-desktop
> sudo apt-get update

2. terminator
> sudo apt-get install terminator
